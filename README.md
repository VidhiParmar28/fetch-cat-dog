
# FetchCatDog 🐱🐶

A beginner-friendly JavaScript project that fetches random cat and dog images 
from public APIs and displays them instantly — no page reload needed.

## Features
- Fetch random cat images (via [Cataas](https://cataas.com))
- Fetch random dog images (via [Dog CEO API](https://dog.ceo/dog-api/))
- Toggle between cats and dogs without refreshing the page
- Built with vanilla JavaScript — no frameworks or libraries

## What I practiced
- `fetch()` and Promises
- `async`/`await`
- DOM manipulation (`createElement`, `appendChild`, `querySelectorAll`)
- Event listeners
- Working with different JSON response shapes across two APIs
